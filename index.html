Raymer Calculations (Excel Tool)

FILE TYPE: .XLSX | CONTEXT: BEng AEROSPACE ENGINEERING

A comprehensive automated design tool built in Microsoft Excel for Aircraft Conceptual Design. This tool facilitates iterative sizing and multivariate analysis based on Raymer's methodology. Key capabilities include:

  • Initial Sizing & Weight Estimation: Automated empty weight and fuel fraction estimation for Heavy Lift UAVs.
  • Performance Analysis: Modules for calculating climb, glide, turning, and landing performance.
  • Aerodynamics & Structure: Lift coefficient analysis and fuselage length sizing.
  • Propulsion & Launch: Motor selection and catapult launch calculations.
